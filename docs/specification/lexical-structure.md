## 3. Идентификаторы

Допустимые символы:
- Буквы: `a`-`z`, `A`-`Z`, `а`-`я`, `А`-`Я`
- Цифры: `0`-`9`
- Символ подчёркивания `_`

Первый символ:
- Не может быть цифрой.

**Идентификаторы НЕ регистро независимые**

### В нотации EBNF
```
identifier = letter , { letter | digit | "_" }
```

## 4. Литералы

### 4.1. Литералы чисел

#### Литералы целых чисел
- Литерал целого числа, состоящий из арабских цифр `0`, `1`, ..., `9`
  
#### Литералы чисел с плавающей точкой
- Литерал десятичной дроби, состоящий из арабских цифр `0`, `1`, ..., `9` и одного разделителя `.` в середине литерала

### 4.2. Литералы строк

- однострочные строковые литералы, пишутся в двойных кавычках: `"Строка"`
- `\\`: обратный слэш
- `\t`: табуляция
- `\n`: новая строка
- `\r`: возврат каретки

## 5. Операторы

### Арифметические операторы

- `приумножу`: инкремент
- `умалю`: декремент
- `+`: сложение/унарный плюс
- `-`: вычитание/унарный минус
- `*`: умножение
- `/`: деление
- `%`: остаток
- `даруй`: присвоение

### Операторы сравнения

- `паче`: больше или равно
- `меньше`: меньше или равно
- `велий`: больше
- `малый`: меньше

### Логические операторы

- `или`: логическое ИЛИ
- `и`: логическое И
- `яко`: логическое равенство
- `негоже`: не равно
- `не`: логическое отрицание

- `истинно`: логическая истина
- `лукаво`: логическая ложь

## 6. Прочие лексемы

- `поклон`: конец конструкции
- `воистину`: начало блока кода
- `аминь`: конец блока кода
- `(`: начало выржения и вызова функции
- `)`: конец выржения и вызова функции
- `,`: разделитель операторов
- `[]`: массивы

## 7. Комментарии

Для коментария используется символ `#`, за которым следует любая последовательность символов, заканчивающаяся концом строки (символом новой строки)
