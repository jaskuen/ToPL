# Типы данных

1. `благодать` — целочисленный тип
2. `кадило` - число с плавающей точкой
3. `словеса` — строковый тип
4. `верующий` - логический тип
5. `ничтоже` - тип, не имеющий значения

## Литералы

Виды литералов:

1. Литерал числа, например: `2025`, `123.456`
2. Литерал строки, например: `"Hello, world!"`

## Операторы

- Для арифметических операторов `+`, `-`, `*`, `/`, `%` операнды должны быть числами. Результат — целое число, если оба числа - целые, либо число с плавающей точкой, если **хотя бы** одно из чисел таковое
- Также оператор `+` применим к строкам, результатом является строка, содержащая конкатенацию двух строк
- Для операторов `приумножу`, `умалю` операндом выступает число. Результат - тот же числовой тип
- Для операторов сравнения `паче`, `меньше`, `велий`, `малый` операнды должны быть числами. Результат - логическая истина или ложь
- Для логических операторов сравнения `или`, `и`, `не` операндами(-ом) выступают(-ет) булевские(-ое) значения(-ие). Результат - булевское значение
- Для операторов сравнения `яко` и `негоже` допускаются любые операнды одинакового типа, включая различающиеся числовые типы данных. Результат - логическая истина или ложь

## Вычисление по короткой схеме

Вычисление по короткой схеме осуществляется для логических операторов `и`, `или`. В этом случае второй (правый) оператор не вычисляется.

## Неявное преобразование типов

Неявное преобразование типов возможно, когда ожидаемый тип - число с плавающей точкой, а получено - целое число. Остальных случаев нет

## Явное преобразование типов

Для явного преобразования типов в язык были добавлены функции, позволяющие перейти от:
- Строк к числовым значениям
- Числовых значений к строкам

## Возврат значений

Способностью возвращать значение в данном языке обладают только функции. Поскольку в языке предусмотрено обязательное объявление типа, функции, объявленные с типом `ничтоже`, не будут возвращать значение, другие же возвращают объявленный тип данных.\
Главная функция программы возвращает любой из предложенных в языке типов данных.
