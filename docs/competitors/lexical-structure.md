# Лексика C++ и C#
Т.к. их лексика довольна похожа, то дальнейшее описание идет для обоих языков. Если оператор уникален, то будет указано, для какого языка.

## Идентификаторы

Допустимые символы:
- Буквы: `a`-`z`, `A`-`Z`
- Цифры: `0`-`9`
- Символ подчёркивания `_`
- Unicode-символы (например, 😀)

Первый символ:
- Не может быть цифрой.
- Допустимы буквы или `_`.

### C++
Ключевые слова запрещены:
`int`, `class`, `return` и т.д. (полный список зависит от стандарта).

### C#
Ключевые слова запрещены, но можно использовать с префиксом `@`:
Пример: `@class`, `@int`.

## Литералы целых чисел
- Литерал целого числа, состоящий из арабских цифр `0`, `1`, ..., `9`
- Шестнадцатиричная запись: начинается с `0x`, затем цифры `0`-`9`, буквы `a`-`f` в любом регистре
- Двоичная запись: начинается с `0b`, заем цифры `0` или `1`
- Разделители разрядов: апостроф `'` как разделитель

### C++
- Восьмиричная запись: начинается с `0`, затем цифры `0`-`7`

## Литералы чисел с плавающей точкой
- Литерал десятичной дроби, состоящий из арабских цифр `0`, `1`, ..., `9` и одного разделителя `.` в середине литерала
- Экспоненциальная форма: Мантисса + `e`/`E` + `экспонента`: `1.23e-4`, `5E+10`

# Элементы структурного программирования
## Блоки кода

- `{`: начало блока кода
- `}`: конец блока кода
- `;`: звершающий оператор

## Ветвления

- `if`: Начало условной конструкции.
- `else`: Альтернативная ветка.
- `switch`: Начало конструкции выбора.
- `case`: Отметка одного из вариантов выбора в `switch`.
- `default`: Вариант по умолчанию в `switch`.

- `(` и `)` - скобки для условия.
- `:` - двоеточие после case и default.

Тернарный оператор:
- `?`: Разделяет условие и первое выражение (true-ветка)
- `:`: Разделяет true-ветку и false-ветку

## Циклы

- `for`: Цикл с инициализацией, условием и инкрементом.
- `while`: Цикл с предусловием.
- `do`: Цикл с постусловием (используется в паре с `while`).
- `break`: Принудительный выход из цикла (или `switch`).
- `continue`: Переход к следующей итерации цикла.

# Средства для ввода/вывода и другие встроенные функции

### C++
Идентификаторы:
- `cout`: объект стандартного вывода
- `cin`: объект стандартного ввода
- `endl`: манипулятор для завершения строки

- Оператор `<<`: вывод в поток
- Оператор `>>`: ввод из потока

### C#
- Есть статический класс `Console`, внутри него методы для ввода/вывода.
- `Cosnole.WriteLine`: вывод с переносом на новую строку
- `Console.ReadLine`: ввод с переносом на новую строку
- `,`: используется для разделения параметров, передаваемых в функции ввода/вывода

# Пользовательские функции, включая параметры и возвращаемые значения

- `void`: указание на отсутсвие типа
- `return`: завершение функции и возврат значения
- Идентифкаторы: имя функции и список параметров через `,` внутри `()` 

# Встроенные типы данных для целых чисел, чисел с плавающей точкой и строк

Целые числа: `int`, `short`, `long`, `char`, `bool`
- C++: `unsigned`, `size_t`, а также `uint_8/16/32_t`
- C#: `uint`, `ushort`, `ulong`, `byte`, `sbyte`, `nint`
  
С плавающей точкой: `float`, `double`
- C++: `long double`
- C#: `decimal`

Строки: `string` (не являются встроенным типом, т.к. используют библиотеки)

# Выражения с операторами
## Арифметические операторы

- `+`: сложение, унарный плюс
- `-`: вычитание, унарный минус
- `*`: умножение
- `/`: деление
- `%`: остаток от деления
- `++`: инкремент
- `--`: декремент
- `=`: присваивание

А также составные операторы присваивания:
`+=`, `-=`, `/=`, `*=`, `%=`

## Логические операторы

- `true`: логическая истина
- `false`: логическая ложь

- `!` (логическое НЕ)
- `&&` (логическое И)
- `||` (логическое ИЛИ)

## Операторы сравнения

- `==`: равно
- `!=`: не равно
- `<`: меньше
- `>`: больше
- `<=`: меньше или равно
- `>=`: больше или равно

# Пользовательские составные типы данных:
## Массивы

- `[`: начало объявления размера массива или индекса.
- `]`: конец объявления размера массива или индекса.
- `new`: ключевое слово для создания нового экземпляра массива.

Для C++
- `delete`: оператор освобождения памяти

## Структуры (записи)
- `struct`: объявление структуры
- Оператор `.`: доступ к полям структуры

### C#
- `record`: объявление записи (C# 9.0+)
- `with`: позволяет задать конкретный параметр `record`

Пример:
```csharp
public record NamedPoint(string Name, int X, int Y);
var p1 = new NamedPoint("A", 0, 0);
var p2 = p1 with { Name = "B", X = 5 };
```
