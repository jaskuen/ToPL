# Список тестов
## Lexer

### Идентификаторы
- [x] Обычное название `variable`
- [x] Начинается с нижнего подчеркивания `_variable`
- [x] Имя содержит цифры в конце `Var123`
- [x] Имя содержит цифры в середине `Var123raV`

### Числовые литералы
- [x] Целое число `123`
- [x] Число с целой и дробной частью `1.23`
- [x] Число c нулевой целой частью, с дробной частью `0.123`
- [x] Число начинается с 0 `0123`

### Строковые литералы
- [x] Строка без экранированных символов `"Приветствую!"`
- [x] Строка с экранированными символами `"Здравствуйте!\n\tПока!\rДо свидания!"`
- [x] Строка из одного символа `"A"`
- [x] Запрещен неэкранированный перенос строки в литеральной строке `"Начало
строки"`
- [x] Экранирование обратного слеша

### Выражения с арифметическими операторами
- [x] Операции сложения / вычитания `1 + 3 - 2`
- [x] Операции умножения / деления `5 * 10 / 25`
- [x] Остаток от деления `10 % 3`
- [x] Инкремент `приумножу var`
- [x] Декремент `умалю var`
- [x] Присвоение `num даруй 6`

### Выражения с операторами сравнения
- [x] Операция сравнения ">" `num1 велий num2`
- [x] Операция сравнения "<" `num1 малый num2`
- [x] Операция сравнения ">=" `num1 паче num2`
- [x] Операция сравнения "<=" `num1 меньше num2`

### Выражения с логическими операторами
- [x] Логическое И `bool1 и bool2`
- [x] Логическое ИЛИ `bool1 или bool2`
- [x] Логическое равенство `bool1 яко bool2`
- [x] Логическое неравенство `bool1 негоже bool2`
- [x] Логическое отрицание `не bool1`
- [x] Логическая истинна `истинно`
- [x] Логическая ложь `лукаво`

### Комментарии
- [x] Комментарий на отдельной строке `#Комментарий`
